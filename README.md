# MiniGif

MiniGif is a lightweight and easy gif resizer as a Mac Menu Bar app.

## Features
- Resizing
- Image Optimization
- Frame Trimming
- Colour Reduction

## Get Started
### Prerequisites
- Must have a mac, with XCode downloaded

### Installation
1. Clone this repo
2. Open the project in XCode
3. Clean the project `Shift-Command-K`
4. Build! `Command-B`
